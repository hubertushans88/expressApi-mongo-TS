# Express-API Boilerplate with typescript
Simple Express JS API with MongoDB

## Using
- ExpressJs
- MongoDB
- [Mongoose](https://mongoosejs.com/) as ODM
- [Joi](https://joi.dev/api/) as validator



## Folder Structure
```
|- src
  |- config
  |- controllers
  |- middlewares
  |- models
  |- routes
  |- services
  |- utils
  |- validations
```
